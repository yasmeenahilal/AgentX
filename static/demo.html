How to Embed the AgentX Chat Widget

This page demonstrates how to embed the AgentX chat widget on your website. You'll see the chat button in the bottom right corner of this page.

Step 1: Get Your Deployment Details

First, create an agent and deploy it using the Embed deployment method in the AgentX dashboard. You'll receive a deployment ID and some embed code.

Step 2: Add the Widget to Your Site

Simply add the following script tag to your HTML:

<script 
    data-agent="your-agent-name" 
    data-deployment-id="your-deployment-id" 
    data-theme="light"
    src="https://your-agentx-url/static/embed.js">
</script>

Configuration Options

The widget can be customized with the following attributes:

  • data-agent - The name of your agent (required)
  • data-deployment-id - Your deployment ID (required)
  • data-theme - Choose between "light" or "dark" theme (optional, defaults to "light")
  • data-position - Position of the widget: "bottom-right", "bottom-left", "top-right", or "top-left" (optional, defaults to "bottom-right")
